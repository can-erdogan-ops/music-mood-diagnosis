# Music Mood Diagnosis 🎧

A little command-line program that "diagnoses" your mood based on what music genre you're listening to right now.

## How it works

1. Run the script
2. Pick a genre by number or type its name (or type `h` for help)
3. The program guesses your mood and gives you a short message
4. Optionally, it opens a YouTube playlist for that genre

## Run it

```bash
python music_mood_diagnosis.py
```

## Example

```
music mood diagnosis

1. Heavy Metal
2. Lo-fi Hip Hop
3. Classical
...

what are you listening to right now? (h for help): 3

genre: Classical
mood:  melancholic

you didnt really feel like talking to anyone today

open a playlist? (y/n): y
```
